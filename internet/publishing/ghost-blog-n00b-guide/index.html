[OUTDATED] Ghost blog n00b guide

Tuesday, March 28, 2017 - Last updated: Thursday, August 23, 2018

Basic setup

curl -L https://ghost.org/zip/ghost-latest.zip -o ghost.zip

unzip -uo ghost.zip -d ghost

cd ghost && npm install --production

Running on production

Install PM2 server

sudo npm install pm2 -g

Configure the app

In your ghost folder create a config.json file with the following content:

{
  "apps": [
    {
      "name": "ghost",
      "script": "npm",
      "args": "start --production"
    }
  ]
}

Start the server

pm2 start config.json
